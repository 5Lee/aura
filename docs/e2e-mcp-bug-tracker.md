# Phase 3 E2E MCP Bug Tracker

## Usage

- 每个发现的 bug 新增一条记录，状态流转：`OPEN -> WAIT_USER -> IN_PROGRESS -> FIXED -> VERIFIED -> CLOSED`
- `WAIT_USER` 表示已询问用户是否修复，等待确认。
- 证据优先记录截图与复现路径。
